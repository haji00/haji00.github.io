My Notes on HTML

<head></head>
The head of the web page
<body></body>
The content of the web page
<header></header>
The header of the web page when used inside the <body></body> tag, and the header of the article when used inside the <article></article> tag.
<footer></footer>
The footer of the web page when used inside the <body></body> tag, and the footer of the article when used inside the <article></article> tag.
<h1></h1>
The most important heading of the page. The next level of headings will be h2,h3,h4,h5, and h6.
<p></p>
To specify a paragraph in the web page.
<image></image>
To inserts an image inside the web page.
<nav></nav>
The navigation of the web page
